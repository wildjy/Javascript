[문서 정의]

web browser : IE8↑, chrome 최신

문서 제목 (title) : 컨텐츠 제목 | 서비스명 (ex: 캐치카페 > 커뮤니티 | 캐치)

DOCTYPE + HEAD

<!DOCTYPE html> <html lang="ko"> <head> <meta charset="UTF-8"> <meta http-equiv="X-UA-Compatible" content="ie=edge"> <title>TITLE</title> <link rel="stylesheet" href="../css/Default2.css" type="text/css"> /* 기본 css link */ </head>

[HTML 작성 규칙]

css파일을 component 별로 link 시켜주기

<link rel="stylesheet" href="../css/list.css"> <ul class="list"> <li><a href="#">내용 내용 내용</a></li> <li><a href="#">내용 내용</a></li> </ul>

하위 tag는 tab 한 개 들여쓰기

<ul> <li><a href="#">menu</a></li> <li><a href="#">menu</a></li> </ul>

table은 col, th, td만 tab 한 개 들여쓰기

<table border="1"> <caption>table 내용</caption> <colgroup> <col style="width:33%"><col style="width:33%"><col> </colgroup> <thead> <tr> <th scole="col">제목</th> <th scole="col">col</th> <th scole="col">col</th> </tr> </thead> <tbody> <tr> <td>내용</td> <td>content</td> <td>content</td> </tr> </tbody> </table>

주석 표기

// Yep <!-- 안내 --> /* 주석 시작 */ <!-- // 안내 --> /* 주석 끝 */ // Nope <!---- 안내 ----> /* -- 는 내용 앞뒤로 두 개만 */ <!--//안내--> /* 내용 앞뒤로 한 칸씩 띄우기 */

[SCSS file]

파일명 : 모두 소문자

_reset.scss : page, popup의 공통 초기화

_helper.scss : 변수, mixin, function

common.scss : page, popup의 공통 style

keyword.scss : 키워드

more.scss : 페이지 더보기

movetop.scss : 상단으로 가기

sns.scss : SNS 공유하기

videodetail.scss : 영상 상세 페이지

videolist.scss : 영상 목록 페이지

default.css : head의 공통 css (common.css, Jinhak의 layout.css, webfont를 import 하기 위해 만듬)

[SCSS naming]

형태 약자 : 레이어-layer / 목록-list / 박스-box / 배너-bn / 버튼-btn / 셀렉트박스-select / 아이콘-ico / 입력폼-input / 탭-tab / 타이틀-tit / 텍스트-txt / 표-tbl /

// Yep .tbl_write /* 형태_의미 */ .list_item1 /* 형태_의미숫자 */ // Nope .btnCom /* 카멜 형식 X */ .tab-dep2 /* -(middle bar) 사용 X */ .ico_sns_1 /* 숫자앞에 _ 사용 X */ .Ck_com1 /* 대문자 사용 X */

.at_경우 : 경우에 따라 다른 타입을 정의

// .at4 - li의 갯수를 4개로 width:25%를 표현할 때 .list { li { float: left; } &.at4 li { width: (100% / 4); } } // .at_youtube - youtube에서 이미지를 불러올 때 .list { ~~~ &.at_youtube { ~~~ } }

.selected : tab메뉴 등에서 선택된 경우

.tab_txt1 { li { &.selected { ~~~ } } }

id는 사용하지 않는다. (layout 제외)

[SCSS 작성 규칙]

// Yep .btn { display: inline-block; padding: 2px 10px; font-family: "Verdana", "Malgun Gothic", sans-serif; } // Nope .btn { display: inline-block; /* 들여쓰기는 tab 한 개 */ } .btn { padding:2px 10px; /* 속성 간 한 칸 띄우기 */ } .btn {display: inline-block; padding: 2px 10px;} /* 한 줄 쓰기 X */ .btn { font-family: 'Verdana', 'Malgun Gothic', sans-serif; /* 따옴표는 큰타옴표 사용 */ }
